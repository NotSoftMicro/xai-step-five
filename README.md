# STEP 5

Elon's 5-step Algorithm as a live Grok agent for Standard Operating Procedures.

### What it does
Paste any SOP. Step Five applies Elon's algorithm to make it shorter, clearer, and more efficient while keeping all necessary tool details.

### How to run
1. pip install streamlit openai python-dotenv
2. Add your Grok API key to .env as GROK_API_KEY=...
3. streamlit run app.py
